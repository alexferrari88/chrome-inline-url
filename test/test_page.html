Link Displayer Extension Testing Page

How to test this extension:
  1. Open Google Chrome and navigate to chrome://extensions/.
  2. Enable Developer mode (toggle in the top-right corner).
  3. Click Load unpacked (top-left) and select the extension directory:
    /home/alex/src/chrome-inline-url
  4. Once loaded, open this page (or drag it into a tab).
  5. Click the extension icon in the toolbar. It should display the current domain (as a local file path or server host).
  6. Click Whitelist Site. By default, you will see inline URLs next to ALL links on the page.
  7. Now, edit the selector in the popup card to: .article-content and press Enter (or click save). Only links inside the white article card should show their inline URLs! Links in the header and sidebar should be clean.
  8. Press Alt + Shift + U to toggle the link displays on/off.
  9. Click the Simulate Dynamic Content button below to verify that new links also get inline URLs.
  10. To test hostname-only formatting:
    • Open the extension popup and toggle the Show just hostname checkbox under Global Link Format. The links on this page (e.g. Hacker News, Reddit Programming) should now only show hostnames (e.g. (news.ycombinator.com), (reddit.com)) instead of full URLs.
    • In the popup site card for this website, change the Format override select to Full URL. Observe that the links on this page revert to full paths (e.g., showing reddit.com/r/programming) even though the global hostname setting is checked.
    • Set the site override back to Use Global Setting to see it sync back with the global setting.
